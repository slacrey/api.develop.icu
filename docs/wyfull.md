# 网易新闻详情解析

> 通过 网易新闻 `docid` , 获取新闻的主体详情解析



## 接口参数

请求地址：`/api/new/detail`

请求方式：`get`  `post`

请求参数

| 参数名称 | 必选 | 类型     | 说明               |
| -------- | ---- | -------- | ------------------ |
| `docid`  | 是   | `string` | 网易新闻的 `docid` |
