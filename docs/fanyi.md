# 翻译查询接口

> 文本翻译查询接口



## 接口参数

请求地址：`api/fanyi`

请求方法：`get`  `post`

请求参数：

| 参数名称 | 必选 | 类型     | 说明                                     |
| -------- | ---- | -------- | ---------------------------------------- |
| `q`      | 是   | `string` | 要翻译的文本                             |
| `from`   | 否   | `string` | 要翻译的文本 语种  默认 `auto`  自动识别 |
| `to`     | 否   | `string` | 目标语种，默认 `en`                      |

**语种列表**

| 语言简写 | 名称         |
| -------- | ------------ |
| auto     | 自动检测     |
| zh       | 中文         |
| en       | 英语         |

| wyw      | 文言文       |
