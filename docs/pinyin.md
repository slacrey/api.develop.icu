# 中文转拼音

> 可以将输入的中文转为拼音。



## 接口参数

请求地址：`/api/pinyin`

请求方法：`get` `post`

请求参数：

| 参数名称 | 必选 | 类型      | 说明                                               |
| -------- | ---- | --------- | -------------------------------------------------- |
| `word`   | 是   | `string`  | 要转换的中文字符                                   |
| `de`     | 否   | `string`  | 拼音分割符，默认为一个空格，分隔符最多为一个字符串 |
| `tone`   | 否   | `boolean` | 拼音是否带音标，默认0。输入值 为 1 或者 0          |
| `abbr`   | 否   | `boolean` | 是否首字母模式，默认0。输入值 为 1 或者 0          |
