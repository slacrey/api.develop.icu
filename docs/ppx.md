# 皮皮虾无水印视频解析

> 根据皮皮虾分享的链接，解析出没有水印的视频



## 接口参数

请求地址：`/api/video/ppx`

请求方法：`get`  `post`

请求参数

| 参数名称 | 必选 | 类型   | 说明                                                        |
| -------- | ---- | ------ | ----------------------------------------------------------- |
| `url`    | 是   | string | 皮皮虾视频的分享地址,例如：`https://h5.pipix.com/s/uW4KXh/` |
