# 名人名言

> 随机返回一条古今中外名人名言



## 接口参数

请求地址：`/api/mingyan`

请求方式：`get`  `post`

请求参数：

| 参数名称 | 必选 | 类型  | 说明                          |
| -------- | ---- | ----- | ----------------------------- |
| `typeid` | 否   | `int` | 返回名言的类型, 范围：`1 -45` |
