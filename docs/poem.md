# 藏头诗生成

> 根据输入的 `4` 字头，生成 5 言或者 7 言藏头诗，根据唐诗训练的 AI藏头诗生成，99% 成功率

## 接口参数

请求地址：`/api/poem	`

请求方式：`get` `post`

请求参数：

| 参数名称  | 是否必选 | 类型     | 说明                                |
| --------- | -------- | -------- | ----------------------------------- |
| `keyword` | 是       | `string` | 藏头诗包含的字，目前支持 `4` 个中文 |
