# 抖音无水印视频解析

> 根据抖音分享的链接，解析出没有水印的视频

## 接口参数

请求地址：`/api/video/dy`

请求方法：`get`  `post`

请求参数

| 参数名称 | 必选 | 类型   | 说明                                                   |
| -------- | ---- | ------ | ------------------------------------------------------ |
| `url`    | 是   | string | 抖音视频的分享地址,例如：`http://v.douyin.com/rJjgX8/` |
