请求地址：`/api/new/wbtop`

请求方式：`get`  `post`

| 参数名称 | 必选 | 类型 | 说明                |
| -------- | ---- | ---- | ------------------- |
| `num`    | 否   | int  | 数据条数，默认 `50` |
