# 历史上的今天

> 获取历史上同一月同一天发生的事情



## 接口参数

接口地址：`/api/eventHitory`

请求方式：`get`  `post`

请求参数：

| 参数名称   | 必选 | 类型  | 说明                                         |
| ---------- | ---- | ----- | -------------------------------------------- |
| `monthday` | 否   | `int` | 月份和日期，如 `1014`,不填默认获取今天的事件 |
